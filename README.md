# energy_intensity_and_decarbonization
Replication code for the paper "Energy Intensity, offshoring and the illusion of decarbonization" (Amadei, Dosi, Pintus, 2026) - Nature Communications

To reproduce our findings, an array of data sources, which are not all publicly available, have to be deployed. 

The value added and GDP data that support the findings of this study are available at the United Nations Statistics Division:
National accounts estimates of main aggregates (2023) http://data.un.org/Explorer.aspx?d=EDATA 

The emissions data for Figure 1 and the energy data used throughout the paper can be found at the International Energy Agency's Greenhouse gas emissions from energy: https://www.iea.org/data-and-statistics/data-product/greenhouse-gas-emissions-from-energy-highlights and World energy balances https://www.iea.org/data-andstatistics/data-product/world-energy-balances respectively.

The production-based and consumption-based emissions data are available from the EORA Multi-Regional Input-Output
Database: Carbon footprint of nations: https://worldmrio.com/footprints/carbon

Finally, the final demand and sectoral composition thereof have been retrieved from the OECD's ICIO tables https://www.oecd.org/en/data/datasets/inter-country-input-output-tables.html

IMPORTANT REMARK: IEA data are not public available at the details employed in the analysis and IEA data, which are licensed, requiring to purchase access that has been done by the authors to perform the analysis of the paper. For this reason, this code cannot be fully replicated without the access to these data nad for this reason we cannot freely share the "Database" folder that the code is referering to. Please use this repository only to access the MATLAB code and our README.doc if you are interested in see how the code have been developed.
